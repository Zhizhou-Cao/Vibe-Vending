# 🎰 Dopamine Vending Machine

> **A Vibe Coding Experiment.** > 将无聊的碎片时间，转化为成就感获得瞬间。



## 📖 简介 (Introduction)

这是一个完全由 **HTML/CSS/JS 单文件** 驱动的 Web 应用，诞生于一次 **Vibe Coding** 的灵感爆发。

在这个数字时代，我们习惯了通过短视频获取廉价的多巴胺。**Dopamine Vending Machine** 试图通过“老虎机”这种经典的随机奖励机制，反向激励用户去完成一些**微小的、健康的、有益身心的任务**。

收集硬币 -> 拉动拉杆 -> 完成随机任务 -> **获得成就感！🎉**

## 🎮 玩法 (How to Play)

1.  **💰 收集金币**：不仅是等待，你需要点击屏幕上随机掉落的金币（致敬 PVZ 风格）来积累财富。
2.  **🎰 拉动拉杆**：集齐 **3 枚硬币**，用力拉下老虎机的拉杆。
3.  **⚡️ 随机挑战**：系统会从知识库中随机抽取一个微任务（可能是冷知识、拼图、深呼吸或运动）。
4.  **✅ 即时反馈**：完成任务后，享受满屏的庆祝特效，给大脑一次健康的“马杀鸡”。

## ✨ 特性 (Features)

* **沉浸式 UI**：暗黑霓虹风格，配合丝滑的 CSS 动画和拟物化拉杆交互。
* **8 大任务系统**：
    * 🧩 **碎片拼图**：简单的逻辑互动。
    * 🃏 **记忆挑战**：激活你的短期记忆。
    * 🌿 **正念呼吸**：跟随动画平复心情。
    * 🧘 **动感时刻**：办公室/居家微运动。
    * 📖 **灵感卡片**：随机读取一条哲学金句或冷知识。
    * 🎵 **听觉 SPA**：精选音乐推荐。
    * 👀 **护眼行动**：强制休息眼睛。
    * 🧹 **微观清理**：整理桌面或数字空间的断舍离。
* **纯静态部署**：无需后端，一个 HTML 文件即可运行，极速加载。
